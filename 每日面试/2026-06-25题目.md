## 2. TypeScript `NoInfer<T>` 与泛型推断控制

- 题型：TypeScript 类型题
- 难度：中等
- 来源或考察方向：[TypeScript Utility Types - NoInfer](https://www.typescriptlang.org/docs/handbook/utility-types.html#noinfertype)
- 题目正文：下面的 API 希望 `defaultValue` 必须属于 `options` 推断出的联合类型。为什么普通泛型可能把非法默认值也纳入 `T`？请使用 `NoInfer<T>` 改写类型，并说明它与类型断言的本质区别。

```ts
declare function createSelect<T extends string>(
  options: readonly T[],
```
