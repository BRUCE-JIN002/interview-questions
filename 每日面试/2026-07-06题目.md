# 2026-07-06 全栈偏前端面试题

## 1. JavaScript Import Attributes 与 JSON Module

- 题型：原理题
- 难度：中等
- 来源/考察方向：[MDN Import attributes](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Statements/import/with)
- 题目：`import data from "./data.json" with { type: "json" }` 解决了什么问题？它与 `fetch()` 加载 JSON 的缓存、执行时机和适用场景有何区别？

- 难度：中等
- 来源/考察方向：[TypeScript 4.4 Release Notes](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-4-4.html#exact-optional-property-types)
- 题目：开启 `exactOptionalPropertyTypes` 后，`foo?: string` 与 `foo: string | undefined` 有什么区别？它为什么会影响 PATCH DTO、对象展开和属性存在性判断？

- 难度：中等
- 来源/考察方向：综合整理

## 4. 浏览器跨源隔离与 SharedArrayBuffer

- 题型：浏览器/安全题
- 难度：困难
- 来源/考察方向：[MDN Cross-Origin-Opener-Policy](https://developer.mozilla.org/en-US/docs/Web/HTTP/Reference/Headers/Cross-Origin-Opener-Policy)

## 5. React Effect 中的请求竞态

- 题型：场景题
- 难度：中等
- 来源/考察方向：[React useEffect](https://react.dev/reference/react/useEffect)
- 题目：搜索关键字连续变化时，旧请求可能晚于新请求返回。如何在 `useEffect` 中避免旧结果覆盖新结果？比较忽略旧响应、`AbortController` 与数据请求库三种方案。

## 6. Vue 3 `watch` 的清理与竞态控制

- 来源/考察方向：[Vue Watchers](https://vuejs.org/guide/essentials/watchers.html)
- 题目：用 Vue 3 `watch` 实现“依赖变化即请求”，要求取消过期请求并避免旧结果回写。说明清理函数何时执行，以及 `flush: "pre" | "post" | "sync"` 的选择。
