# 2026-06-24 每日 15 道全栈偏前端面试题

## 1. JavaScript 闭包与内存释放

- 题型：八股 + 场景题
- 难度：中高级
- 可能来源或考察方向：综合整理；词法作用域、闭包、垃圾回收、内存泄漏
- 题目正文：

一个 React/Vue 页面卸载后，Chrome Memory 面板仍能看到该页面的大量数据对象。请解释闭包为什么可能延长对象生命周期，并说明如何判断这是正常保留还是内存泄漏。请至少分析事件监听器、定时器、缓存和异步回调四类常见来源。

## 2. TypeScript：`satisfies`、类型注解与类型断言

- 题型：类型系统 + 代码分析
- 难度：中高级
- 可能来源或考察方向：TypeScript 官方文档 `satisfies`：https://www.typescriptlang.org/docs/handbook/release-notes/typescript-4-9.html
- 题目正文：

请比较下面三种写法在类型校验、字面量类型保留和不安全断言方面的差异，并说明业务配置对象更推荐哪一种。

```ts
type RouteConfig = Record<string, {
  path: `/${string}`;
  auth: boolean;
}>;

const routes1: RouteConfig = {
  home: { path: '/home', auth: false }
};

const routes2 = {
  home: { path: '/home', auth: false }
} as RouteConfig;

const routes3 = {
  home: { path: '/home', auth: false }
} satisfies RouteConfig;
```

## 3. 手写带并发上限的异步任务池

- 题型：手写实现
- 难度：中高级
- 可能来源或考察方向：综合整理；Promise、并发控制、顺序保持、失败隔离
- 题目正文：

请实现：

```ts
async function asyncPool<T, R>(
  limit: number,
  items: T[],
  worker: (item: T, index: number) => Promise<R>
): Promise<PromiseSettledResult<R>[]>
```

要求最多同时执行 `limit` 个任务，结果顺序与输入顺序一致，单个任务失败不阻塞其他任务；并说明如何增加 `AbortSignal` 取消能力。
