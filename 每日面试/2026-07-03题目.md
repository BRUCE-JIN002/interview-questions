# 2026-07-03 全栈偏前端面试题

## 1. JavaScript `Proxy` 不变量与可撤销代理

- 题型：八股文 / 场景题
- 难度：高级
- 来源或考察方向：[MDN Proxy](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Proxy)、[MDN Proxy.revocable](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Proxy/revocable)
- 题目正文：状态库用 `Proxy` 拦截对象读写。为什么代理的 trap 不能任意伪造不可配置属性、不可扩展对象或只读数据属性的结果？`Reflect` 在 trap 中有什么价值？请设计一个组件卸载后立即失效的可撤销代理，并说明它与冻结对象、访问器属性的差异。

## 2. TypeScript 条件类型的分发与关闭分发

- 题型：类型设计 / 手写题
- 难度：中高级
- 来源或考察方向：[TypeScript Conditional Types](https://www.typescriptlang.org/docs/handbook/2/conditional-types.html)
- 题目正文：解释 `T extends U ? X : Y` 在 `T` 为裸类型参数且传入联合类型时为何会分发。实现 `ToArray<T>` 与不分发版本 `ToArrayNonDist<T>`，比较它们对 `string | number` 的结果；再说明 `never`、`any` 和 `unknown` 参与条件类型时容易造成哪些误判。

## 3. 手写带 `leading`、`trailing`、`cancel`、`flush` 的 debounce

- 题型：手写实现 / JavaScript
- 难度：中高级
- 来源或考察方向：综合整理；定时器、闭包、`this` 与返回值语义
- 题目正文：实现 `debounce(fn, wait, options)`，支持首调用立即执行、尾调用执行、取消等待以及立即刷新；保留原函数的 `this` 和参数。请明确 `leading` 与 `trailing` 同时开启时单次调用的行为、`cancel()` 后的状态，以及异步函数返回值应如何约定。

## 5. HTTP `103 Early Hints` 与关键资源加载
