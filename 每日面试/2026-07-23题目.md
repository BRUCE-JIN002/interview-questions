# 2026-07-23 全栈偏前端每日面试题

## 1. JavaScript：`AbortSignal` 的组合、超时与取消语义

- 题型：原理 + 手写
- 难度：中等
- 可能来源或考察方向：[MDN AbortSignal](https://developer.mozilla.org/docs/Web/API/AbortSignal)
- 题目：搜索页需要在新输入到来、组件卸载或 5 秒超时时取消 `fetch`。如何用 `AbortSignal.any()`、`AbortSignal.timeout()` 组合取消来源？浏览器不支持时怎样实现最小降级？为什么取消不等于服务端一定停止执行？

## 2. TypeScript：`const` 类型参数与字面量推断

- 题型：原理 + 类型设计
- 难度：中等
- 可能来源或考察方向：[TypeScript 5.0 const Type Parameters](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-5-0.html#const-type-parameters)
- 题目：设计 `defineRoutes()`，使传入的路由数组保留精确的 `path` 字面量，并能推导出路径联合类型。`const T` 类型参数与调用点 `as const` 分别解决什么问题，何时仍需要运行时校验？

## 3. ES6+：迭代器提前退出与资源清理

- 题型：原理 + 场景
- 难度：较难
- 可能来源或考察方向：[MDN Iteration protocols](https://developer.mozilla.org/docs/Web/JavaScript/Reference/Iteration_protocols)
- 题目：一个自定义 generator 持有分页请求和 WebSocket 订阅，消费者在 `for...of` 中 `break` 或抛错时，如何确保资源释放？说明 generator 的 `finally`、`return()` 与异步迭代器的 `for await...of` 分别如何参与清理。

## 4. 浏览器：Shadow DOM 事件委托与跨边界事件

- 题型：原理 + 排障
- 难度：中等
- 可能来源或考察方向：[MDN Event.composedPath](https://developer.mozilla.org/docs/Web/API/Event/composedPath)
- 题目：页面在 document 上做点击事件委托，但组件迁移到 Shadow DOM 后，`event.target` 不再指向内部按钮。解释事件重定向、`composed`、`composedPath()` 和 open/closed shadow root 的差异；怎样设计既可复用又不泄漏内部实现的事件契约？

## 5. 网络：Service Worker 缓存与“离线可用但数据不陈旧”

- 题型：场景设计
- 难度：较难
- 可能来源或考察方向：[web.dev Workbox caching strategies](https://web.dev/articles/workbox-caching-strategies-overview)
- 题目：新闻列表要支持弱网和离线阅读，但登录态、权限变更和突发内容必须尽快生效。如何区分 Cache First、Network First、Stale While Revalidate？如何设计缓存 key、版本切换、失效通知和登出清理，避免把私有接口响应缓存给错误用户？

## 6. React：受控输入、延迟值与昂贵列表渲染

- 题型：性能优化 + 场景
- 难度：中等
- 可能来源或考察方向：[React useDeferredValue](https://react.dev/reference/react/useDeferredValue)
- 题目：筛选输入框驱动一个 5 万行的本地列表，输入出现卡顿。为什么不应延迟受控 input 的 value？如何用 `useDeferredValue`、`memo` 和虚拟列表让输入保持即时，同时向用户表达“结果正在追赶输入”？它与防抖请求有什么本质差异？

## 7. Vue 3：`shallowRef`、不可变数据与大对象更新

- 题型：原理 + 性能优化
- 难度：中等
- 可能来源或考察方向：[Vue Reactivity API: Advanced](https://vuejs.org/api/reactivity-advanced.html#shallowref)
- 题目：地图实例、图表实例和服务端返回的大型树数据放进 Vue 状态后出现无意义的深层代理开销。`shallowRef`、`markRaw`、`triggerRef` 各适合什么场景？如果树数据采用不可变更新，应该如何触发视图更新而不破坏第三方实例？

## 8. 手写：并发受限、可取消的异步任务池

- 题型：手写实现
- 难度：较难
- 可能来源或考察方向：综合整理
- 题目：实现 `runPool(tasks, { concurrency, signal })`：`tasks` 是返回 Promise 的函数数组；最多并发 `concurrency` 个；结果与输入顺序对应；某个任务失败时停止派发新任务；外部取消时拒绝并避免未开始任务执行。说明已开始任务为何只能协作式取消。

## 9. 算法：滑动窗口的最短子数组

- 题型：算法题
- 难度：中等
- 可能来源或考察方向：[LeetCode 209](https://leetcode.com/problems/minimum-size-subarray-sum/)
- 题目：给定全为正整数的数组 `nums` 和目标值 `target`，求和大于等于 `target` 的最短连续子数组长度；不存在返回 0。给出 O(n) 解法，说明左右指针移动的单调性；若数组允许负数，为什么该方法失效？

## 10. Next.js：动态路由参数、404 与缓存边界

- 题型：框架工程 + SEO
- 难度：中等
